Contributing to Open Track

Thank you for your interest in this project!


How to Contribute
- Fork this repository
- Create a new branch for your changes
- Make your changes and test them
- Submit a pull request with a short description of what you changed


Guidelines
- Keep code clean and add comments where needed
- Use f-strings for string formatting
- Make sure the script runs from top to bottom without errors
  

Reporting Issues

If you find a bug, open an issue and include:
- What went wrong
- Steps to reproduce it
- Your Python version
