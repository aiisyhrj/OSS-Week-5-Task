Open Track - 
A Python program that manages contributors and tracks issues for an open source project.

What it does
- Register contributors and store their info
- Track issues with priority and status
- Save reports to text and CSV files

How to run
- Open OSS_Week5_Task.ipynb in Google Colab
- Run all cells from top to bottom
- Enter the required info when prompted

Requirement
- Python 3
